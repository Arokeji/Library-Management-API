# Library API

Ready to use with Jest tests.
Routes included in Postman.

Project based on:
- Node.js using TypeScript and MongoDB

Included in this project:
- Express
- Swagger
- Bcrypt
- Cors
- Dotenv
- JsonWebToken
- Mongoose
- MySQL2
- Reflect-metadata
- Multer
- TypeOrm
- Validator
- Eslint & Prettier
